# Submission Checklist

Sebelum *final handover*, pastikan berikut lengkap:

- [ ] **PPT Final** dengan template resmi (isi: Biodata, Hasil, Insight, Rekomendasi)
- [ ] **Link GitHub repository** (repo ini)
- [ ] **Link Video Presentasi** (YouTube/Drive)
- [ ] (Opsional) Link folder file hasil pengerjaan (contoh: SQL exports, PDF dashboard)

Tambahan praktik baik:
- [ ] README menjelaskan cara replikasi (sudah disediakan)
- [ ] Semua SQL tervalidasi (no error)
- [ ] Screenshot *quality checks* lulus
- [ ] Tautan dashboard Looker Studio (mode *view* publik / akses reviewer)
